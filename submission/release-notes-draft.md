# ClaimGPT Release Notes Draft

## Version

`0.1.0`

## Summary

Initial public submission of ClaimGPT, a ChatGPT app for insurance claim triage.

## Included

- Read-only claim analysis tool
- Read-only claim workspace render tool
- ChatGPT widget for structured triage review
- Public privacy and support pages
- Container-based deployment support

## Notes For Review

- This version is a submission-safe starter focused on read-only triage workflows.
- The app does not create, update, or submit claims externally.
- The tool annotations reflect that current scope.
